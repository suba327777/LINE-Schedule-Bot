##  どんなアプリか

普段からGoogleCalendarを利用しており、もっと手軽に予定を見れたらなと思い、このbotを開発しました。
ただ、このbotはサービスアカウントを用いて開発しているため、このbotを利用しようと考えている方は別途GCPでサービスアカウントを作成する必要があります。
